Project Scope:
(a) Develop an automation solution using Docker, Ansible, NETCONF or a combination
of these technologies. The project must be developed collaboratively using Git and/or
GitHub.

(b) The automation solution must perform the following device configuration tasks:
Configure IP address, user account, banner message, interface description, static route
and retrieve device information

(c) Automatically collect and display the following Linux system information:
hostname, current date and time, CPU information, memory usage, disk usage, loggedin users, top 5 running processes by CPU usage.
